IntelIntent Mutation Kit - Execution Instructions

This kit will help you reset your PC and configure it as a development environment for IntelIntent.

Execution Order:
1. Run the PowerShell script in '1_Backup' to back up your system.
2. Run the script in '2_System_Wipe' to clean your system.
3. Follow the instructions in '3_Reinstall_Windows' to reinstall Windows.
4. After reinstalling, run the script in '4_Environment_Configuration' to set up the environment.
5. Run the script in '5_IntelIntent_Seeding' to seed IntelIntent modules.
6. Run the script in '6_Visual_Dashboard_Setup' to create the dashboard.
7. Run the script in '7_Mutation_Confirmation' to verify the mutation.

Ensure you run each step in order and verify completion before proceeding to the next.
