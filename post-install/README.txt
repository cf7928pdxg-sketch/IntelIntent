IntelIntent Post-Install Package

Instructions:
1. After reinstalling Windows, copy the entire 'IntelIntent_PostInstall' folder to your user directory.
2. Run 'IntelIntent_Mutation.ps1' using PowerShell (Right-click > Run as Administrator).
3. Optionally, place 'Launch_IntelIntent_Reset.bat' in your Startup folder:
   C:\Users\<YourUsername>\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup
4. This will seed the IntelIntent environment and log progress in 'mutation_log.txt'.
