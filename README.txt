IntelIntent_PC_Reset README

This folder contains PowerShell scripts and recursive components to reset and mutate your PC into the IntelIntent development environment.

Each phase folder includes:
- A PowerShell script to perform basic operations
- A Recursive_Operations subfolder with a checkpoint file

Phases:
1. Backup
2. System_Wipe
3. Reboot
4. Environment_Configuration
5. IntelIntent_Seeding
6. Visual_Dashboard_Setup
7. Mutation_Confirmation
8. Recovery_Logs

To execute the mutation pipeline, run each PowerShell script in order.
